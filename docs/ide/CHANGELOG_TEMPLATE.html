Enso Developer Documentation

Next Release

This update contains major performance improvements and exposes new privacy user settings. We will work towards stabilizing it in the next weeks in order to make these updates be shipped in a stable release before the end of the year.


Visual Environment

  • You can now launch missiles directly from the GUI. It was technically possible since version 3.0.0-alpha.7, but it was never exposed as a button.
  • The graph editor stops shaking and running away when you are pasting JavaScript code.

Runtime

  • The JiT compiler got new optimizations and produces code up to 10x faster than C++.
  • You do not need to keep your computer in -20°C to prevent the engine from crashing anymore.

Libraries

  • The new JSON library allows you to parse 2Gb files in 50ms.
  • The Regexp library exposes now methods to test the expressions directly on humans.
